#GitHubの使い方まとめ  


##GitHubからコピーする
```
$ git clone [URL]  
Cloning into 'hoge'...  
$ ls  
hoge  

$ cd hoge  
$ ls  
README.md  
$ vim README.md  
$ git add README.md  
$ git commit -m "commit hoge"  
 1 file changed,   
$ git push  
```


##リポジトリの更新  
```  
$git pull origin master  
```
